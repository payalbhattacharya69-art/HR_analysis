📊 HR Analytics Dashboard (Power BI)

📌 Project Overview
This project presents an interactive HR Analytics dashboard developed using Power BI. It focuses on analyzing employee attrition, workforce demographics, and salary trends to generate actionable business insights.

🎯 Objectives
- Analyze employee attrition and retention patterns
- Identify key factors influencing attrition
- Understand workforce demographics (age, education, salary)
- Support HR decision-making with data-driven insights

🛠️ Tools Used
- Power BI

📁 Dataset
- HR dataset containing employee details such as age, salary, education, attrition status, and years of experience

📊 Key Insights
- Total Employees: 1470 | Attrition: 237 | Attrition Rate: 16.1%
- Majority attrition occurs among employees aged 25–35
- Employees with lower salary bands show higher attrition
- Average employee age is 37 years
- Average tenure is around 7 years
- Certain education fields show higher attrition trends

🚀 Features
- KPI cards for quick insights (Attrition, Avg Salary, Age, etc.)
- Attrition analysis by age and education
- Interactive visuals for better exploration
- Clean and user-friendly dashboard design
